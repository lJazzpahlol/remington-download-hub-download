# Remington Download Hub

Public download repository for Remington Download Hub, the single installer for
the Remington Accessibility app family.

## Download

Latest Windows package:

https://github.com/lJazzpahlol/remington-download-hub-download/releases/latest/download/RemingtonDownloadHub-win64.zip

## Install

1. Download the zip.
2. Right-click it and choose **Extract All**.
3. Open the extracted folder.
4. Double-click `INSTALL.bat`.
5. Launch **Remington Download Hub** from the Desktop or Start Menu.
6. Click **CHECK ALL**, then install or reinstall Nova, Panelz, REMsound,
   Scrollz, and Starboard from inside the Hub.

The Hub creates `config.json` and `remhub.log` beside the app after first run.
Those files are local to your machine.

No accounts. No telemetry. One download, then the Hub installs the individual
apps.
